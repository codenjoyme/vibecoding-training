# LnD Migration Guide

## Formatting Conventions

I expect that everything (except terms) enclosed in `` ` `` or ` ``` ` will be transferred to the Learn platform as a code container with copy functionality.

This means:
- Inline code `like this` → rendered as an inline code element with copy support.
- Code blocks 
``` 
like 
this
``` 
→ rendered as a full code block with a copy button.

> **Note:** Terms (e.g. product names, tool names) that appear in backticks for styling purposes only — and are not meant to be copied as code — should be reviewed case by case.
