# Module 12 — Completion Task

Congratulations on completing Module 12: AI Skills / Tools Creation!

To verify your work, copy the prompt below and paste it into the same AI assistant session where you completed the module. The agent will collect the data and produce a report file. Once the report is generated, copy the contents of `work/module-12-report.md` and paste them into the answer text field on the learning platform.

```
# Module 12 — AI Skills / Tools Creation: Completion Report

You just completed Module 12 of the AI-assisted development training. Now generate a completion report so the autocheck system can verify your work.

## Instructions

Collect the following data from your workspace and produce a report.

### Data to collect

1. **Instruction file contents:** Paste the full contents of the `.instructions.md` (or `.agent.md`) file you created.

2. **Script file contents:** Paste the full contents of the corresponding script file (e.g. Python `.py`, Node.js `.js`, or shell `.sh`) that the instruction references or automates.

3. **Script execution test:** Run the script and capture the output:
   ~~~
   python <your_script>.py --help
   ~~~
   or the equivalent command for your language. Paste the output.

4. **Brief description:** In 1–2 sentences, describe what the skill/tool does and why you created it.

## Report format

Produce a file called `work/module-12-report.md` with this structure:

~~~markdown
# Module 12 Completion Report

## Skill Description
<1–2 sentence description of what the tool does>

## Instruction File
- Filename: <instruction filename>

<paste full instruction file contents>

## Script File
- Filename: <script filename>
- Language: <Python / JavaScript / Shell / etc.>

<paste full script file contents>

## Script Execution Output
<paste output of running the script with --help or a test invocation>
~~~

Save this report as `work/module-12-report.md`.
```
