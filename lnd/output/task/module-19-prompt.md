```
# Module 19 — GitHub Coding Agent Delegation: Completion Report

You just completed Module 19 of the AI-assisted development training. Now generate a completion report so the autocheck system can verify your work.

## Instructions

Collect the following data and produce a report.

### Data to collect

1. **GitHub Issue:** Provide the URL of the issue you created for the coding agent to work on, along with its title and description.

2. **Pull Request:** Provide the PR URL or number that the coding agent created (or that was created as a result of delegation).

3. **Delegated task description:** In 2–3 sentences, describe what task you delegated and why it was a good candidate for agent delegation.

4. **Outcome assessment:** Did the agent complete the task correctly? What did you review or adjust?

## Report format

Produce a file called `work/module-19-report.md` with this structure:

~~~markdown
# Module 19 Completion Report

## GitHub Issue
- URL: <issue URL>
- Title: <issue title>
- Description: <brief description of the issue>

## Pull Request
- URL: <PR URL>
- PR Number: <#number>
- Status: <Open / Merged / Closed>

## Delegated Task
<2–3 sentences describing what you delegated and why>

## Outcome Assessment
- Agent completed correctly: <Yes / Partially / No>
- What you reviewed or adjusted: <description>
~~~

Save this report as `work/module-19-report.md`.
```
