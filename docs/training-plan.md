# Vibecoding for Managers - Training Plan

## Course Philosophy

This training is highly practical. Each video provides a new actionable skill that managers can immediately apply to their work. The progression builds from basic setup to advanced integration, enabling participants to go from zero to building functional prototypes with AI assistance.

---

## Module Breakdown

### Module 1.1: Installing VSCode + GitHub Copilot
**Duration:** 5-7 minutes  
**Skill:** Set up your first AI coding environment

**Topics:**
- Installing Visual Studio Code
- GitHub Copilot extension installation and authentication
- Basic configuration and settings
- First interaction with Copilot
- Keyboard shortcuts and UI overview

**Outcome:** Working VSCode environment with GitHub Copilot ready to use

---

### Module 1.2: Installing Cursor
**Duration:** 5-7 minutes  
**Skill:** Set up alternative AI-native IDE

**Topics:**
- What is Cursor and why use it
- Installation process
- Key differences from VSCode
- Initial configuration
- When to use Cursor vs VSCode

**Outcome:** Cursor IDE installed and configured

---

### Module 2: Model Selection
**Duration:** 5-7 minutes  
**Skill:** Choose the right AI model for your task

**Topics:**
- Overview of available models (GPT-4, Claude, etc.)
- Model strengths and weaknesses
- Cost vs capability tradeoffs
- Matching models to task types
- Switching between models

**Outcome:** Understand when to use which model

---

### Module 3: Agent Mode & AI Mechanics
**Duration:** 5-7 minutes  
**Skill:** Understand how AI coding assistants work

**Topics:**
- What is agent mode
- How AI processes your requests
- Context windows and limitations
- Tool use and function calling
- Why AI makes certain decisions

**Outcome:** Mental model of AI assistant behavior

---

### Module 4: Effective Prompting
**Duration:** 5-7 minutes  
**Skill:** Iterate prompts instead of arguing with AI

**Topics:**
- Why arguing doesn't work
- Going back and editing your prompt
- Progressive refinement technique
- Being specific vs being vague
- Common prompting mistakes

**Outcome:** Ability to refine prompts for better results

---

### Module 5: Version Control with Git
**Duration:** 5-7 minutes  
**Skill:** Track changes with baby steps approach

**Topics:**
- Why version control matters in vibecoding
- Baby steps methodology
- Committing after each successful change
- Rolling back when AI makes mistakes
- Git basics for non-developers

**Outcome:** Use Git to safely iterate with AI

---

### Module 6: Custom Instructions
**Duration:** 5-7 minutes  
**Skill:** Create instructions for AI assistants

**Topics:**
- What are custom instructions
- When to use them
- Meta-skill: instructions for creating instructions
- Examples of effective instructions
- Storing and reusing instructions

**Outcome:** Write effective custom instructions

---

### Module 7: Learning from Hallucinations
**Duration:** 5-7 minutes  
**Skill:** Use AI mistakes to improve instructions

**Topics:**
- What are hallucinations
- Why they happen
- Using mistakes as feedback
- Updating instructions based on errors
- Pattern recognition in AI failures

**Outcome:** Turn AI errors into instruction improvements

---

### Module 8: AI Skills - Context, Tools & Memory
**Duration:** 5-7 minutes  
**Skill:** Understand AI's eyes, hands, and memory

**Topics:**
- Eyes: What AI can see (tools)
- Hands: What AI can do (tools)
- Memory: What AI remembers (instructions/context)
- How these affect AI capabilities
- Introduction to Skills concept

**Outcome:** Understand AI assistant architecture

---

### Module 9: Model Context Protocol (MCP)
**Duration:** 5-7 minutes  
**Skill:** Connect AI to any data source

**Topics:**
- What is MCP
- Why it's a game changer
- How to connect AI to databases, APIs, files
- Finding and using MCP servers
- Real-world MCP examples

**Outcome:** Understand MCP and its potential

---

### Module 10: Development Environment Setup
**Duration:** 5-7 minutes  
**Skill:** Install essential tools for vibecoding

**Topics:**
- Installing Node.js
- Docker basics and installation
- Package managers (npm, pnpm)
- Terminal basics for Windows/Mac
- Verifying installations

**Outcome:** Complete development environment ready

---

### Module 11: Rapid Prototyping
**Duration:** 5-7 minutes  
**Skill:** Build a PoC in 2 hours

**Topics:**
- Scoping a prototype with AI
- Breaking down requirements
- Letting AI generate initial structure
- Iterative development with AI
- From idea to working demo

**Outcome:** Methodology for rapid prototyping with AI

---

### Module 12: QA with Chrome DevTools MCP
**Duration:** 5-7 minutes  
**Skill:** Test web applications with AI assistance

**Topics:**
- Chrome DevTools MCP server setup
- Using AI to debug frontend issues
- Automated testing with AI
- Inspecting elements and network requests
- AI-powered QA workflow

**Outcome:** AI-assisted quality assurance process

---

### Module 13: Advanced MCP Integration
**Duration:** 5-7 minutes  
**Skill:** Add MCP interface to your application

**Topics:**
- Why add MCP to your app
- Creating custom MCP server
- Controlling your app from IDE
- Real-time debugging and manipulation
- Advanced integration patterns

**Outcome:** Ability to create MCP-enabled applications

---

## Hands-on Project

After completing all modules, participants will build a complete application that demonstrates:
- Web interface
- Automation scripts
- MCP integration
- Proper version control
- Quality assurance

**Expected Duration:** 4-6 hours  
**Evaluation:** AI-powered automated assessment

---

## Learning Path

```
Setup → Understanding → Techniques → Tools → Integration → Project
  ↓         ↓              ↓          ↓          ↓           ↓
1-2    →   3-4      →     5-7    →  8-10   →  11-13    → Final
```

Each module builds on previous ones, creating a complete skill set for effective vibecoding.
