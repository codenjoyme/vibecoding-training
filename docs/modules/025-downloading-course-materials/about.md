## Prerequisites

- **Module 010**: Installing VSCode + GitHub Copilot

## Why Download Locally?

Working with course materials locally in your IDE provides several advantages:

- **Offline access** - Work through modules without internet connection
- **IDE integration** - Use AI assistance while learning
- **Hands-on practice** - Edit and experiment with examples directly
- **Self-contained learning** - No dependency on external platforms
- **Version control ready** - Can later convert to Git repository (Module 060)

## What You'll Get

The repository contains:

- All training modules with theory and walkthroughs
- Code examples and exercises
- Scripts and tools for automation
- Configuration templates
- Practice projects and assignments
