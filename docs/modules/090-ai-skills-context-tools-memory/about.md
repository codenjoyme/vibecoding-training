# AI Skills - Context, Tools & Memory

**Duration:** 5-7 minutes  
**Skill:** Understand AI's eyes, hands, and memory

## Topics

- Eyes: What AI can see (context)
- Hands: What AI can do (tools)
- Memory: What AI remembers (limitations)
- How these affect AI capabilities
- Introduction to Skills concept

## Learning Outcome

Understand AI assistant architecture

## Prerequisites

- Solid understanding of AI mechanics
- Experience with custom instructions
