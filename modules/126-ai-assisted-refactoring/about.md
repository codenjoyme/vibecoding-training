# AI-Assisted Refactoring

**Duration:** 15 minutes

**Skill:** Use AI as a refactoring partner to clean up messy AI-generated code through safe, incremental baby-step improvements backed by Git.

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- Recognising signs of messy AI-generated code
- Safe refactoring with Git: one change, one commit, verify, repeat
- AI refactoring prompts: extract function, simplify logic, rename for clarity
- Applying 3 refactoring passes to a real PoC codebase
- Creating a reusable refactoring instruction with code style rules

## Learning Outcome

You can identify refactoring candidates in AI-generated code, apply incremental improvements using AI without breaking functionality, and have a reusable refactoring instruction for consistent code quality.
