# AI Study Buddy — Learning New Tech

**Duration:** 15 minutes

**Skill:** Use AI as a structured learning partner to rapidly understand any unfamiliar technology through the Feynman method — explain, quiz, practice, verify

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- The Feynman technique adapted for AI-assisted learning
- Structured "Study Buddy" prompt for concept exploration
- Active recall: AI-generated quizzes to test understanding
- Practice tasks: hands-on exercises with AI evaluation
- Knowledge mapping with Mermaid diagrams
- Building a reusable study instruction for any technology

## Learning Outcome

You will learn a real unfamiliar technology in a single session using AI as your tutor. You will create a reusable "study buddy" instruction file that automates the learning workflow for any topic — from initial explanation through quizzes and practice to a visual knowledge map of what you've covered.
