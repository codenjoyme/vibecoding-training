# AI for Data Analysis & Reporting

**Duration:** 15 minutes

**Skill:** Analyse any CSV or tabular dataset through AI conversation and generate a structured written report — without writing pandas code from scratch or opening Excel.

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- Using AI as an analyst: asking questions about raw data in natural language
- Generating pivot tables, grouped summaries, and computed metrics through AI
- Creating visualisations (matplotlib charts, Mermaid diagrams) from data
- Asking AI to write summary reports with conclusions and recommendations
- Building a reusable Python analysis script that runs on any CSV file

## Learning Outcome

You can take any CSV export — from Jira, test results, usage logs, or a spreadsheet — and turn it into structured insights and a written report using AI, without needing data science skills or prior pandas knowledge.
