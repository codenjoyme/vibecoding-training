# Database Schema Design with AI

**Duration:** 15 minutes

**Skill:** Design a relational database schema, generate SQL migrations, and write queries through AI conversation — from plain-language requirements to a working data layer.

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- Describing data models in plain language and getting AI to generate schemas
- Reviewing and iterating on relationships, indexes, and constraints
- Generating SQL migration files (up/down) with AI
- Writing complex queries through AI assistance
- Adding a database layer to an existing PoC project

## Learning Outcome

You can go from "I need to store users with roles and their activity log" to a working, properly constrained SQL schema with migrations — in a single AI conversation — and integrate that schema into your PoC.
