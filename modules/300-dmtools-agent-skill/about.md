# DMtools — Agent Skill & Automation Catalog

**Duration:** 15-20 minutes

**Skill:** Install a third-party agent skill, explore a catalog of 152+ automation tools across 16 integrations, and choose a practical task to accomplish using AI-assisted configuration.

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- What agent skills are and how they extend your AI assistant
- Installing the DMtools skill via one-line command or manually (all platforms)
- The full catalog of DMtools capabilities: 152+ MCP tools, 23 Jobs, JS Agents
- The core DMtools architecture: `{ "name": ..., "params": { input + output } }`
- Choosing and exploring a specific capability from the catalog
- Getting the AI to generate a working configuration for your chosen task

## Learning Outcome

You will be able to install any third-party agent skill into your AI IDE (Cursor or VS Code with GitHub Copilot). You'll have a mental map of all DMtools capabilities organized by category, understand the universal config pattern, and complete a self-chosen exploration task using the installed skill.

## When to Use

- When a teammate shares a new agent skill with you
- When you find a useful skill on [agentskills.io](https://agentskills.io) or GitHub

- When you want to automate repetitive work with Jira, Azure DevOps, Confluence, Teams, Figma, or GitHub
