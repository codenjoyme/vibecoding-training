# Visual Context with Screenshots

**Duration:** 5-7 minutes  
**Skill:** Share visual context with AI using screenshots to get accurate help when interface differs from expectations

## Topics

- Taking screenshots on different operating systems
- Sharing full screen vs partial screen captures
- When to use visual context instead of text descriptions
- Pasting screenshots directly into AI chat
- Practical example: Opening browser console with AI guidance

## Learning Outcome

Use screenshots to show AI exactly what you see, enabling accurate assistance when software interfaces change or differ from AI's knowledge

## Prerequisites

- Completed Module 030 (Model Selection)
- Active AI assistant (VSCode with GitHub Copilot or Cursor)
- Understanding of Agent Mode
- Any web browser installed

## When to Use

- Software interface has changed in newer version
- AI suggests options you don't see in your UI
- Debugging visual layout or styling issues
- Getting help with unfamiliar interface elements
- Verifying AI understands your current screen state
- When text descriptions are insufficient
