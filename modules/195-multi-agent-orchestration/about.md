# Multi-Agent Orchestration

**Duration:** 15 minutes

**Skill:** Design and run multi-agent pipelines where specialised AI agents handle different roles — coder, reviewer, tester — and you orchestrate them to produce higher-quality results than any single agent can achieve alone.

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- Why single agents plateau and multi-agent patterns overcome their limitations
- Defining specialised agent roles with separate instruction files
- Sequential, parallel, and supervisor orchestration patterns
- Building a two-agent coder + reviewer pipeline in VS Code
- Combining local Copilot agents with remote GitHub coding agents for parallel work

## Learning Outcome

You can split a complex task across multiple specialised AI agents, coordinate their outputs, and get results that are more reliable and thorough than what a single agent with a single instruction produces.
