# Translate Any Website with AI Agent

**Duration:** 10-15 minutes
**Skill:** Use Chrome DevTools MCP to translate any website into any language via AI agent commands

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- Using Chrome DevTools MCP for website translation
- AI model-powered text extraction, translation, and DOM injection
- Automating browser navigation with AI agent
- Handling translation edge cases (dynamic content, code snippets, long pages)
- Taking screenshots to verify translation results

## Learning Outcome

AI agent translates any website to any language on command, using Chrome DevTools MCP to navigate and verify the result in a real browser.

## Prerequisites

### Required Modules

- [130 — AI-Powered QA with Chrome DevTools MCP](../130-chrome-devtools-mcp-qa-emulation/about.md)

### Required Skills & Tools

- Chrome DevTools MCP server configured and running
- VS Code or Cursor IDE with MCP integration enabled
- Internet connection

## When to Use

- You need to quickly read a foreign-language website
- You want to test how your site looks in another language
- You are doing localization QA and need to compare translations
- You want to demonstrate browser automation capabilities to a team
