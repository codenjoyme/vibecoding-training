# AI-Assisted Code Review

**Duration:** 15 minutes

**Skill:** Review AI-generated pull requests effectively — using AI to summarise changes, flag problems, and enforce quality — so you can merge with confidence.

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- Why AI-generated code still requires human review and what to look for
- Using Copilot to summarise and analyse pull requests
- Building a code review instruction for structured, consistent feedback
- Identifying common "looks good but isn't" patterns in AI-generated PRs
- Establishing a lightweight team review protocol for AI-assisted development

## Learning Outcome

You can review a pull request created by an AI coding agent — using AI to assist the review — and produce structured, actionable feedback that ensures both correctness and maintainability.
