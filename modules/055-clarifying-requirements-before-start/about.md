# Clarifying Requirements with AI

**Duration:** 5-7 minutes  
**Skill:** Get AI to help you define unclear tasks through questions

**👉 [Start hands-on walkthrough](walkthrough.md)**

## Topics

- When to use this technique (empty project, unclear feature requirements)
- Writing minimal 3-4 sentence initial prompt
- Adding "ask me additional questions before we start"
- Answering AI questions iteratively
- Using "are there any more questions?" to continue clarification
- Building complete requirements through dialogue

## Learning Outcome

Ability to transform vague ideas into actionable plans through AI-driven questioning

## Prerequisites

### Required Modules

- [050 — Effective Prompting](../050-effective-prompting-without-arguing/about.md)

### Required Skills & Tools

- Understanding of how to write precise prompts with statements
- Basic experience with AI coding assistants

## When to Use

- Starting a new empty project without clear plan
- Adding a new feature to existing project but unclear where to start
- Having general idea but unable to articulate detailed requirements
- Needing help breaking down complex tasks
